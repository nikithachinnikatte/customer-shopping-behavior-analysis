# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, PostgreSQL, and Power BI to identify purchasing trends, customer segments, and revenue drivers.

The goal is to derive insights that help understand customer purchasing patterns and improve business decision making.

---

## Dataset

Customer Shopping Behavior Dataset

Key attributes:

* Customer ID
* Age
* Gender
* Item Purchased
* Purchase Amount (USD)
* Review Rating
* Subscription Status
* Discount Applied
* Shipping Type
* Previous Purchases

---

## Tools Used

* Jupyter-Python (Pandas, Matplotlib, Seaborn)
* pgadmin4-PostgreSQL (SQL analytics queries)
* Power BI (interactive dashboard)

---

## Analysis Performed

### Python Analysis

* Data cleaning and preprocessing
* Feature engineering (age groups, purchase frequency)
* Exploratory data analysis
* Correlation analysis

### SQL Analysis

Key queries include:

* Revenue generated by gender
* Customers spending above average despite discounts
* Top 5 highest rated products
* Average purchase amount by shipping type
* Subscription vs non-subscription revenue comparison
* Customer segmentation using CTE
* Revenue contribution by age group

### Dashboard

A Power BI dashboard was created to visualize:

* Revenue by category & sales
* Purchase trends
* Customer segments
* Age group spending patterns
* Shipping Type prefernces

---
## Project Architecture

Data Source (CSV Dataset)
→
Data Cleaning & Exploration (Python - Pandas)
→
Data Analysis Queries (PostgreSQL)
→
Data Visualization (Power BI)
→
Business Insights


## Key Insights

* Subscribers generate higher revenue compared to non-subscribers.
* Loyal customers contribute significantly to total revenue.
* Certain products rely heavily on discount-based purchases.
* Customers aged 25–40 represent a major revenue segment.
